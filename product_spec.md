# Product Spec: PDF Translate Extension (MVP)

## Цель продукта
Создать локально запускаемый продукт для перевода PDF-файлов через Chrome Extension popup и backend на FastAPI с интеграцией OpenAI API.

## Ценность для пользователя
- Быстро перевести PDF без ручного копирования текста.
- Получить готовый скачиваемый файл результата.
- Понимать статус обработки на каждом этапе.

## User Flow
1. Пользователь открывает popup расширения.
2. Выбирает исходный язык (`Auto Detect` или конкретный).
3. Выбирает целевой язык.
4. Загружает PDF (drag-and-drop или кнопкой).
5. Extension отправляет файл и языки в backend.
6. Backend создаёт job, отправляет PDF в OpenAI.
7. Пользователь видит прогресс и текущую стадию.
8. После завершения скачивает результат.

## Основные функции MVP
- Popup UI (MV3) с понятной визуальной иерархией.
- Валидация файла (тип PDF, размер до лимита).
- Запуск перевода через `POST /api/translate`.
- Polling статуса через `GET /api/jobs/{job_id}`.
- Скачивание результата через `GET /api/download/{job_id}`.
- Отображение стадий: `uploaded`, `sending_to_openai`, `processing`, `assembling_result`, `completed`, `failed`.

## Вне scope первой версии
- OCR.
- Chunking длинных документов.
- Сложная реконструкция layout исходного PDF.
- Поддержка форматов кроме PDF.
- Авторизация и история переводов.

## Ограничения v1
- Сканированные PDF могут не переводиться корректно.
- Таблицы и сложные блоки могут терять структуру.
- Очень большие файлы обрабатываются дольше.
- В первой версии результат формируется как markdown-файл (`translated.md`) для надёжности пайплайна.
